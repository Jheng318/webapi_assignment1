There are 5 functions from the node module
**searchPostsByTitle(title)**: a function to search through the list of posts from the database.

**createPost(title, description, comments)**: a function to create a new post passing in the title, description and comments as the parameters.

**addCommentsToPost(postId, comment)**: a function to push a comment into the comments of a selected post

**signUp(username, email, password)**: a function to create a account by passing in the username, email and password into the parameters

**upvotedownvote**: a function that will prompt the user 3 questions to upvote or downvote the selected post
